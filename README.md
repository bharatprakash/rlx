
# MLX

RLX: Single file implementations of Deep Reinforcement Learning algorithms in Apple's [MLX](https://github.com/ml-explore/mlx) Library 


## Installation
```
pip install -r requirements.txt
```

## Quickstart
```
python reinforce.py
```
